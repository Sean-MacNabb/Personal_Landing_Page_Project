Dungeons & Dragons Character Creation

This project was created to simulate the full character creation flow for a Dungeons & Dragons player in a streamlined digital experience. The goal was to guide users through choices such as class, race, and other core setup steps in a way that felt intuitive and easy to navigate. Building it gave me experience translating a multi-step rules-based process into a clearer frontend workflow for the user. It is also a project I plan to revisit and refine as I continue growing my skills and improving the overall experience.

Money Math

Dungeon Dice Roller

This project was built as a utility tool for tabletop fantasy games that rely on dice-based mechanics. The goal was to accept user-entered dice notation, parse the input correctly, and return both the roll results and the total output in a useful format. Through this project, I gained experience working with user input parsing, structured logic, and transforming text-based commands into functional application behavior. It was a strong exercise in creating a focused tool that balances technical problem solving with a straightforward user experience.
